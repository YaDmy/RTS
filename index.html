Правила игры

Моя игра представляет собой 3d лабиринт. Цель - дойти до космческого корабля. Лабиринт состоит из различных препятствий, между которыми есть точки сохранения в виде розовых монет. Игра разделена на 3 уровня, для перехода на следующий уровень необходимо найти жёлтую монету.



Игровой персонаж - робот. управление осуществляется мышью и кнопками W S A D. Для прыжка необходимо нажать пробел.



В случае падения персонаж возвращается на точку последнего сохранения. Для выхода из игры необходимо нажать кнопку выхода на экране, либо воспользоваться комбинацией клавиш alt + F4.



Приятной игры!
